# AI 프롬프트

이 카테고리는 다른 AI 시스템(이미지 생성기·영상 생성기·LLM)에 입력되는 프롬프트를 다룬다. 결과물이 직접 사람 눈에 가지 않고 다른 시스템을 거치므로 (1) 어휘가 그 시스템의 학습 데이터에 맞춰져야 하고, (2) 우선순위와 거절 시나리오가 명시돼야 하며, (3) 엣지 케이스를 미리 차단해야 한다.

## 매핑 표

각 행의 RED·SILVER·BLUE는 5-Color Harness 페르소나 정체성에 해당한다. RED 이성(논리·근거·결론 타당성), SILVER 분야 전문가(베테랑이 첫눈에 짚는 디테일), BLUE 공감(타깃 모델·최종 사용자가 받게 되는 결과물의 신뢰·일관성). AI 프롬프트 카테고리에서 BLUE는 "프롬프트의 결과물을 받는 사람이 다시 돌릴 필요 없이 한 번에 합격으로 받는가"를 본다. 합격선이 표기되지 않으면 9.5다.

| 과제 | RED 이성이 보는 자리 | SILVER 분야 전문가가 보는 자리 | BLUE 공감이 보는 자리(결과물 신뢰) | 합격선 |
|------|---------------------|------------------------------|---------------------------------|--------|
| 이미지 생성 프롬프트 | 컨셉·주제의 명확성·인과 정합성 | 비주얼 디렉터 베테랑이 보는 구도·시각 디테일 | 동일 프롬프트를 세 번 돌려도 일관되게 받는 사용자의 신뢰 | 9.5 |
| 영상 생성 프롬프트 | 장면 구성·서사의 인과 정합성 | 영상 감독 베테랑이 보는 시간 흐름·컷·카메라 워크·조명 | 한 컷이 의도한 호흡대로 생성돼 후속 편집 부담이 없는 신뢰 | 9.5 |
| LLM 시스템 프롬프트 | 행동 규정·역할의 인과 정합성 | LLM 엔지니어 베테랑이 보는 일관성·우선순위 | 엣지 케이스·일반 케이스 모두 거절·역할대로 응답해 운영자가 다시 손대지 않는 신뢰 | 9.5 |

## 합격선 기준 장면

- 이미지 생성 프롬프트. "동일 프롬프트를 세 번 돌려도 컨셉·구도가 일관되게 나온다."
- 영상 생성 프롬프트. "한 컷이 의도한 시간 흐름·카메라 워크대로 생성된다."
- LLM 시스템 프롬프트. "엣지 케이스에서도 거절 시나리오대로 응답하고, 일반 케이스에서는 역할 한 줄대로 일관되게 응답한다."

## 문체·금지 뱅크

### 이미지 생성 프롬프트
- 문체. 주제, 구도, 스타일, 광원, 톤의 다섯 단. 그 분야 종사자 어휘 (앵글, 렌즈, 조명).
- 분량. 50~300자. Midjourney는 짧게(50~150자), DALL-E는 좀 더 길게(150~300자).
- 금지. "멋진", "예쁜" 같은 모호 형용사. 스타일 키워드 다섯 개 이상.

### 영상 생성 프롬프트
- 문체. 한 프롬프트 한 장면. 카메라 워크 명시. 시간 흐름 적기.
- 분량. 100~500자. 한 컷이면 100~200자, 시퀀스면 300~500자.
- 금지. 추상적 무드 표현. 컷 길이 부재.

### LLM 시스템 프롬프트
- 문체. 역할 한 줄, 행동 규정 셋, 거절 시나리오 명시. 우선순위 순.
- 분량. 200~2000자. 단순 역할이면 200~500자, 복잡한 에이전트면 1000~2000자.
- 금지. "좋게 답해줘" 류 모호 지시. 엣지 케이스 무시.

## fallback 5단계 합성 알고리즘

1. **출력 모달리티를 본다.** 이미지 / 영상 / 텍스트 / 오디오 / 멀티모달 중 무엇인가.
2. **타깃 시스템을 본다.** Midjourney / DALL-E / Sora / Veo / GPT / Claude / 자체 모델 중 무엇인가.
3. **두 후보를 고른다.** 모달리티 일치 + 시스템 일치가 1순위.
4. **문체 합성.** 핵심 의도 + 그 시스템 어휘 + 우선순위 + (텍스트면) 거절 시나리오.
5. **금지 합성.** 모호 형용사 + 엣지 케이스 무시 두 줄을 기본 금지로.

예. **"음악 생성 프롬프트(Suno·Udio)"**. 모달리티 오디오. 시스템 Suno/Udio. 후보 = 영상 생성 프롬프트(시간 흐름) + 이미지 생성 프롬프트(스타일 어휘). 합성. 장르·BPM·악기·보컬 톤·가사 후크 다섯 단. 금지. 모호 무드 + 길이 부재.
